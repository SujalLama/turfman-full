## Instructions to start the project

### To start backend
- Please install postgres and node to start the backend
- Steps:
    - `cd backend`
    - `npm install`
    - `npm run develop`
- This will start the server at localhost:1337

### To start frontend
- Please install node to start the frontend
- Steps
    - `cd frontend`
    - `npm install`
    - `npm run dev`
